## 本项目需要依赖 SDL2
需要首先安装 SDL2

```bash 
sudo apt install libsdl2-2.0
sudo apt install libsdl2-dev
sudo apt install libsdl2-mixer-dev
sudo apt install libsdl2-image-dev
sudo apt install libsdl2-ttf-dev
sudo apt install libsdl2-gfx-dev
```
